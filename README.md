# Simulateur de Cerf-Volant 3D

Simulation physique interactive d'un cerf-volant avec moteur physique réaliste.

## Exécution locale

**Prérequis:** Node.js

1. Installer les dépendances:
   `npm install`
2. Lancer l'application:
   `npm run dev`
